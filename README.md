## What is this project about

The goal is simple, I will get the voice input from user like "தக்காளி ஒரு கிலோ" i have to parse this string and have to extract meaningful information like following

input: "தக்காளி ஒரு கிலோ"
output:
{
"product": "தக்காளி",
"quantity": "1",
"unit": "கிலோ"
}

Next thing is to identify the actual product from the list

I will have list of products from there i have to find the relevent product
